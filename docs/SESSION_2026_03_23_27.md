## Currency System Documentation

### Like Economy
- **Daily likes:** 20 free (resets midnight), 50 for premium
- **Bonus likes:** Max 10, earned by clearing incoming like queue
- **Premium likes:** 2/day, requires subscription, bypasses queue-full and age-range checks

### Queue System
- Users who liked you appear first in feed (priority 1-3)
- 10+ qualified likes pending → `must_process_all` flag forces processing
- Target has 10+ pending likes → your like requires premium ("queue full")
- Liker outside target's age range → requires premium ("age range")
